# 租号酷（Zuhaoku）

游戏账号分时租借平台

## 快速开始

### 部署

1. 配置环境变量（参考 [ENV_CONFIG.md](./ENV_CONFIG.md)）
2. 部署到 Zeabur / Railway / Docker
3. 完成

### 环境变量

项目完全通过环境变量配置，所有配置说明请参考 [ENV_CONFIG.md](./ENV_CONFIG.md)

## 技术栈

- Spring Boot 3.2 + MyBatis-Plus + MySQL
- Vue 3 + TypeScript
- 支付宝开放平台 SDK
